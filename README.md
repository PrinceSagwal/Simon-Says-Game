
# 🎮 Simon Says Game

A simple browser-based implementation of the classic **Simon Says Game** built using **HTML, CSS, and JavaScript**. The game challenges players to memorize and repeat an increasingly long sequence of colored button flashes.

## 🚀 Features

- 🎯 Interactive gameplay
- 🎨 Four colored game buttons
- ✨ Button flash animations
- 📈 Level progression
- 🧠 Random sequence generation
- ✅ User input validation
- ❌ Game Over detection
- 🔄 Restart game by pressing any key

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ▶️ How to Play

1. Clone this repository:

```bash
git clone https://github.com/PrinceSagwal/Simon-Says-Game.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

4. Press any key to start the game.

5. Watch the flashing button carefully.

6. Repeat the sequence by clicking the buttons.

7. The sequence grows by one color every level.

8. Clicking the wrong button ends the game.
